# Student Performance Analyzer

A Python-based tool to analyze academic performance of undergraduate students
using credit-weighted grading systems.

## Motivation
Students often receive raw marks but lack insight into strengths, weaknesses,
and overall academic standing. This project provides a transparent and
customizable performance analysis.

## Features
- CGPA calculation
- Subject-wise performance analysis
- CSV-based input
- Clean terminal report

## Tech Stack
- Python 3

## How to Run
```bash
python analyzer.py
